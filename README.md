## Student Housing Management System (Laravel Backend)

This is a university group project focused on building a Student Housing Management System. My role in this project was the backend development using Laravel.

### Project Overview
The system is designed to manage student accommodation efficiently, including student registration, financial tracking, and handling administrative operations related to housing services.

### My Role (Backend – Laravel)
I was responsible for developing the backend APIs and core business logic using Laravel, ensuring the system is scalable, secure, and well-structured.

### Key Features
- **Student Management:** Registering students and managing their personal and housing information within the system.  
- **Payment Tracking:** Monitoring student payments, recording transactions, and tracking outstanding balances.  
- **Violations Management:** Logging and managing student violations related to housing rules.  
- **Complaints System:** Handling student complaints and providing a structured way to track and resolve them.  
- **API Development:** Building RESTful APIs to connect the backend with the frontend application.  

### Technologies Used
- Laravel Framework (PHP)  
- RESTful API Architecture  
- MySQL Database  

### System Design
The backend was designed with a modular structure to separate concerns such as authentication, student management, payments, complaints, and violations. This ensures maintainability and scalability for future enhancements.

### Purpose
The goal of this project was to simulate a real-world student housing management environment and apply backend development principles, including API design, database relationships, and business logic implementation.

### Summary
This project demonstrates practical experience in backend development using Laravel, focusing on building secure, scalable, and efficient systems for managing student housing operations.
